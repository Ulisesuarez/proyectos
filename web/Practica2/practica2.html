XHTML (Lenguaje de Marcado de Hipertexto Extensible) es una versión más estricta y limpia de HTML , que nace precisamente con el objetivo de remplazar a HTML ante su limitación de uso con las cada vez más abundantes herramientas basadas en XML. XHTML combina la sintaxis de HTML, diseñada para mostrar datos, con la de XML, diseñado para describir los datos.

XHTML, está orientado al uso de un etiquetado correcto y por tanto exige una serie de requisitos básicos a cumplir en lo que a código se refiere para que posea una estructuración coherente:

¿Qué necesita mi documento para ser XHTML?
